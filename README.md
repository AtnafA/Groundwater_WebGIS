AAWSA_gDB is a modular and integrated WebGIS java application which designed and implemented to collect, process, manage and share groundwater data. AAWSA_gDB is implemented by tailoring MVC REST API and Springboot java framework. Moreover, the system also intgrated spatial data processing and interoperability management tools such as GeoServer and QGIS.

integrates  modular system, AAWSA designed and developed modular groundwater-data management system (Fig. 4) that has four components (distinct layers) such as Presentation Layer (user interface), Service Layer, Data Access Layer (DAL) and Centralized Groundwater Database Server (AAWSA-gDB). The presentation layer holds components such as a login page, output module, and input module. The service layer implements functions such as connection refuse and permit. The data access layer (DAL) implements model object that contains modules such as data processing, analysis, and common service modules. The fourth layer, AAWSA-gDB database repository is designed using optimized and normalized relational database query.
